Aviseringar

Denna kategori av krav har att göra med hur applikationen aviserar användaren om olika händelser som användaren kan definiera samt frågeformulärer som forskarna vill skicka åt användarna.

Aviseringar via e-mail

Description
Användaren ska kunna välja en e-postadress dit aviseringar skickas.
Rationale

Detta ger användaren flera alternativ att bli informerad av vad som pågår i lokalpolitiken utan att behöva kolla applikationen flera gånger.

Aviseringar på enheter

Description
Aviseringen ska vara synlig på samma enhet som applikationen är installerad på (telefon, tablet).
Rationale

Detta gör applikationen mer lättanvänd från slutanvändarens perspektiv.

Note
Detta krav har en koppling till Plattformar kravet.
